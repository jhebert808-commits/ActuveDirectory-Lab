# User and Group Management Lab

This section covers creating, managing, and organizing users and groups in an Active Directory environment.  
These labs demonstrate how identity, access, and permissions are controlled in a Windows domain.

## Topics Covered

- Creating user accounts in Active Directory Users and Computers (ADUC)
- Creating and managing security groups
- Adding users to groups
- Understanding group scopes (Domain Local, Global, Universal)
- Understanding group types (Security vs Distribution)
- Organizing users and groups with Organizational Units (OUs)
- Applying permissions using groups (AGDLP / AGUDLP models)
- Resetting passwords and unlocking accounts
- Disabling, enabling, and deleting accounts

-## Purpose of This Lab

User and group management is one of the most important responsibilities of a sysadmin.  
These labs help build skills for:

- Managing access to resources
- Applying least privilege
- Troubleshooting login and permission issues
- Maintaining a clean and organized domain structure
- Supporting users in a real IT environment

More details will be added as I build out each part of the lab.
