# Troubleshooting Logs

This section contains detailed notes on issues I encountered while building my Sysadmin Lab, along with the steps I took to diagnose and resolve them.  
Troubleshooting is a core skill for any sysadmin, and this folder documents my real-world problem‑solving process.

## What This Folder Includes

- Error messages and screenshots
- Hypotheses and possible causes
- Steps taken to test each theory
- What worked and what didn’t
- Final resolution
- Notes for preventing the issue in the future
- Patterns I noticed while troubleshooting

## Why Troubleshooting Matters

Troubleshooting builds muscle memory and the skills for IT work

- Diagnosing issues under pressure  
- Understanding how systems behave  
- Learning from mistakes  
- Building intuition for future problems  
- Documenting fixes for myself, others and future reference  

This folder shows my ability to think critically, stay persistent, and solve problems independently.

More logs will be added as I continue building my lab.
