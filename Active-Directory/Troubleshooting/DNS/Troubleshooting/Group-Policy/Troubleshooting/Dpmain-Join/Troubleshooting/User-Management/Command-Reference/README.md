# Command Reference Index

This section contains a quick lookup list of all commands I use in my Sysadmin Lab.  
Each command is grouped by category with a short description.  
Detailed explanations are available in the subpages inside this folder.

## Windows Commands
- ipconfig /all
- ipconfig /flushdns
- ipconfig /registerdns
- gpupdate /force
- gpresult /r
- w32tm /resync /force

## Networking Commands
- ping
- nslookup
- tracert

## Active Directory Tools
- dsa.msc
- gpmc.msc
- dnsmgmt.msc
- eventvwr.msc
